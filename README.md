## Tecnologias
- Python, Pytest, Requests, Selenium, GitHub Actions

## Instalação e execução

### API
```bash
cd api-tests
pip install -r requirements.txt
pytest tests/ -v
```

### Web

```bash
cd web-tests
pip install -r requirements.txt
pytest tests/ -v
```
